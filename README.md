<h1>EasyFinder Front-End</h1>
Repositorio do Front-End do EasyFinder um aplicativo movel de Controle e Auxilio de Itens Perdidos/Encontrados

<h2>Instruções</h2>
<p>• para rodar essa aplicação clone este repositorio; <br>
• ao clonar entre no diretorio raiz e abra o terminal nesta pasta <br>
• em seguida instale as seguintes dependencias: <br>
</p>

```typescript
yarn install
```

<p>• Após isso rode o servidor expo: <br>
</p>

```typescript
npx expo start
```
